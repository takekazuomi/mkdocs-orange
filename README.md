Runs MkDocs server with any directories
======================================================================

install
----------------------------------------------------------------------

1. copy `orange.sh` in your $PATH.
1. setup "${XDG_DATA_HOME:-$HOME/.local/share}/orange"
  ```
  cp -r share "$XDG_DATA_HOME/orange"
  ```

run
----------------------------------------------------------------------

1. run `orange.sh add`
1. run `orange.sh up`
1. open `localhost:8000` in your browser
